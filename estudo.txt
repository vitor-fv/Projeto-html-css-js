     ANOTAÇÕES SOBRE MEU CÓDIGO

- Html é case insensitive (não importa se for maiusculo ou minusculo).
- a tag <title>: denomina o nome da minha página web.
- a tag <h1>: define a hieraquia maior dos titulos no corpo do texto (h1-h6).
- a tag <p>: cria um paragráfo.
- a tag <br>: faz a quebra de linha do texto.

- id: É o atributo para identificar elementos (não pode se repetir o mesmo identificador).
- class: É o atributo para identificar elementos (pode repetir em diversos elementos ou várias classes no 
mesmo elemento).

- a tag <style>: é onde coloco o meu css dentro do html.
- o uso do #name: serve para dar estilo aos elementos com id.
- o uso do .name: serve para dar estilo aos elementos com classe.

- Semântica é usar a tag correta, no local correto.
- a tag <b> = bold (Negrito) e <strong> é significado de *MAIS FORTE*.
- a tag <i> = italic (itálico) e a <em> é siginificado de enfâse (amoooooo).

- tag <a> é a tag que faz o link para sua página.
- O atributo "href" indica aonde vai ser direcionado quando clicado no link.
- Se eu quiser abrir uma janela em outra guia/tela, posso usar o elemento "_blank".
- *Estudar mais sobre  os atributos e elementos do link!*

- a tag <img src> é a tag para gerar ou  inserir imagens na tela.
- O atributo alt é usado como descrição da imagem. Ele é obrigatório, pois ele diz resumido da imagem.

- a tag <del> serve para  indicar algo que foi excluído.
- a tag <ins> serve para indicar algo que foi adicionado.
- a tag <s> serve  para destacar alguma palavra ou frase.
- a tag <u> serve para sublinhar alguma coisa.

- a tag <small> serve para deixar o texto menor.
- a tag <sup> serve para colocar algo acima do texto base.
- a tag <q> serve para colocar algo entre aspas.
- a tag <blockquote> serve para criar blocos de citações e o atributo cite  serve para dar referência da fonte.
- a tag <hr> serve para criar uma linha horizontal. tem que ser colocada fora  das tags de fechamento.
- a tag <code> serve para colocar código fonte. fonte: monospace.
- a tag <pre> serve para manter a preformatação do texto e ela não pode ser colocada na tag p.
    
Tags de contâiner genéricos : div, span, section, article, aside
- a tag <span> serve para agrupar varios elementos semelhantes (inline).
- a tag <div> serve  para separar blocos (block)
